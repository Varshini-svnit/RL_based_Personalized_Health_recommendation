# RL-Based Personalized Health Recommendation System

A healthcare recommendation framework that combines Reinforcement Learning, Thompson Sampling, Kalman Filtering, and probabilistic LLM filtering for personalized adaptive interventions.

---

## Features

- Thompson Sampling for adaptive decision-making
- Kalman Filter-based reward estimation
- Probabilistic LLM filtering
- Personalized healthcare recommendations
- Experimental evaluation and visualization

---

## Technologies Used

- Python
- Google Colab
- NumPy
- Pandas

- Transformers
